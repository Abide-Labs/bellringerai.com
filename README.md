# BellRingerAI.com

Static GitHub Pages landing page for BellRinger, the Abide Labs AI BDC for dealerships.

## Local preview

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Domain

Custom domain is set by `CNAME`:

```text
BellRingerAI.com
```

DNS should point the apex domain to GitHub Pages and `www` to the GitHub Pages host.
